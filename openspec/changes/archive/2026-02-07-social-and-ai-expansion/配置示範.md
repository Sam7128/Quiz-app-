# Gemini API 金鑰配置指南

本指南將引導您如何獲取並在 MindSpark 中配置 Gemini API 金鑰，以啟用 AI 解題小助手功能。

## 1. 獲取 API 金鑰

使用 Gemini API 需要 API 金鑰。您可以免費建立一個來開始使用。

1.  前往 [Google AI Studio](https://aistudio.google.com/app/apikey)。
2.  點擊 **Create API key**。
3.  複製生成的金鑰。**請務必妥善保管，不要洩露給他人。**

## 2. 在 MindSpark 中配置

1.  開啟 MindSpark 應用程式。
2.  進入「設定」或「AI 指引」頁面。
3.  找到 **API 金鑰配置** 欄位。
4.  貼上您剛才複製的金鑰。
5.  點擊「儲存」。金鑰將會安全地儲存在您的瀏覽器本地存儲 (LocalStorage) 中。

## 3. 使用 AI 解題小助手

1.  在進行測驗時，點擊題目下方的「詢問 AI」按鈕。
2.  您可以直接詢問有關該題目的疑問，例如：「為什麼選 B 而不是 C？」或「請詳細解釋這個概念」。
3.  AI (預設使用 Gemma 3 / Gemini 3) 將會根據題目背景為您提供解答。

---

*注意：本應用程式直接與 Google Gemini API 通訊，您的 API 金鑰僅儲存在您的裝置上，不會上傳到我們的伺服器。*
