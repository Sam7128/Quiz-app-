# Skill Effects Engine Spec

## Overview
遊戲級技能特效渲染引擎，負責小技能 (5/10/15/20/25 連擊) 的視覺效果。

## Requirements

### REQ-1: 多層粒子動畫
- 技能觸發時產生 8-16 個粒子向外擴散
- 粒子顏色根據技能元素 (火/冰/雷/虛空/神聖/宇宙) 變化

### REQ-2: 漸層光環效果
- 中心光環從 0% 擴展至 200% 後消失
- 光環使用 blur + gradient 實現柔和效果

### REQ-3: 螢幕震動
- 高級技能 (advanced+) 觸發時添加螢幕震動
- 震動幅度: 2-5px，持續 200-400ms

### REQ-4: 技能名稱動態顯示
- 技能名稱從底部滑入，帶漸層背景
- 顯示 2 秒後淡出

## Acceptance Criteria

- [ ] 5 連擊觸發基礎技能，顯示粒子和光環
- [ ] 20 連擊觸發高級技能，添加螢幕震動
- [ ] 動畫使用 GPU 加速 (transform)，FPS 穩定 60
