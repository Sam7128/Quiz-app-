# Battle Mode Bugfix

## Problem Statement
用戶回報戰鬥模式存在四個 Bug：
1. 隕石技能 (meteor_strike) 圖片無法顯示
2. 20-25 題後一擊秒殺過於頻繁
3. 骷髏巫師造型太細太矮
4. 第 35、45 題錯誤觸發大招（應只在 30、40、50 等里程碑觸發）

## Proposed Solution
修復技能動畫類型、調整技能觸發邏輯、擴展傷害護盾機制、增加骷髏巫師視覺比例。

## Impacted Capabilities
- 技能動畫系統
- 戰鬥傷害計算
- 怪物視覺呈現

## Success Criteria
- 隕石技能動畫正確顯示圖片
- 技能只在 5, 10, 20, 30, 40, 50 連擊觸發
- 高連擊時不會一擊秒殺普通/精英怪物
- 骷髏巫師明顯比主角更高大
